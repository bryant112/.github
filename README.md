# bryant112 Account Defaults

Default GitHub community health files for repositories owned by `bryant112`.

This repository provides:

- default issue forms for bug, feature, chore, and spike intake
- a default pull request template
- tracker reference docs for labels, setup, bootstrap, and baseline memory

These files are used by `bryant112` repositories that do not define their own issue templates or pull request template.

Relevant docs:

- `docs/tracker/labels.md`
- `docs/tracker/project-setup.md`
- `docs/tracker/tracker-operating-model.md`
- `docs/tracker/bootstrap-playbook.md`
- `docs/tracker/current-baseline.md`
